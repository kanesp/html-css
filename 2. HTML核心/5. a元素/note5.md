# a元素
超链接

## href属性
hyper refrence: 通常表示跳转地址

1. 跳转地址
2. 跳转到某个锚点（锚链接）

id属性值：全局属性， 表示元素在文档中的唯一编号

3. 功能连接
点击后出发某个功能

-执行js代码, javascript代码
```html
<a href="javascript:javascript代码">
</a>
```
-发送邮件， mailto
要求计算机有安装邮件发送软件：exchange

-拨打电话， tel
要求计算机有安装拨号软件或者使用移动端

## target属性
表示跳转窗口位置
target取值：
- _self：在当前的页面窗口中打开， 默认值
- _blank: 在新的空白窗口打开
